## 二、Claude 第一轮：方案评审

### 2.1 发现的严重问题

#### 🔴 严重：F407 Flash 分区未对齐 STM32 扇区边界

**问题分析**：
- F407ZGT6 Flash 结构：4×16KB + 1×64KB + 7×128KB = 1MB
- 扇区 11：0x080E0000–0x080FFFFF（128 KB）
- v1.0 方案中"参数/标志区 32 KB"起始 0x080F8000，位于扇区 11 **中间**
- "App 备份区" 0x08080000–0x080F7FFF 跨越扇区 8–10 + 扇区 11 一部分
- **后果**：OTA 擦写会破坏相邻区域，导致升级失败

**修正方案**：
```
0x08000000 - 0x0801FFFF   Bootloader    128 KB
0x08020000 - 0x0807FFFF   App A         384 KB
0x08080000 - 0x080DFFFF   App B         384 KB
0x080E0000 - 0x080FFFFF   参数/标志     128 KB
```

### 2.2 其他问题清单

| 类别 | 问题 | 建议 |
|------|------|------|
| 重要 | 未指定 F103 具体型号 | 明确 C8T6（64KB Flash） |
| 重要 | 电机电源与 MCU 共地风险 | 加解耦电容 + 编码器信号双绞线 |
| 重要 | LVGL 内存估算紧 | 字体放 W25Q64 动态加载 |
| 重要 | 没有安全兜底 | IWDG + 软启动 + 通信超时停机 |
| 重要 | TJA1050 与 3.3V 电平 | 优先选 SN65HVD230 |
| 重要 | CAN 超时阈值未给 | 100ms 离线 / 200ms 停机 |
| 优化 | OTA 阶段应提前 | 放到 Phase 2，先做最小 Bootloader |
| 优化 | 建议加 FreeRTOS | F407 上多任务裸机撑不住 |
| 优化 | 12V 2A 偏小 | 建议 12V/5A |
| 优化 | 缺电源/地隔离 | π 型滤波 + 大电容 |

### 2.3 电源建议

**结论**：12V 适配器 + DC-DC 模块 是正确思路。

**推荐架构**：
```
12V/5A 适配器
  ├─ 12V 直供 BTS7960 VM（电机）
  └─ 12V → 5V DCDC（LM2596 或 MP1584）
           ├─ 5V → BTS7960 VCC
           ├─ 5V → TJA1050 VCC
           ├─ 5V → 屏幕背光
           └─ 5V → 板上 AMS1117 → 3.3V
                   ├─ F407 / F103 / Flash / EEPROM
```

**关键器件**：
- 12V/5A 开关电源适配器
- 12V→5V DCDC 模块（LM2596，3A）
- 电机入口 1000μF/25V + 0.1μF 解耦
- 不推荐 12V→5V 用 LDO（散热太大）

### 4.4 关键引脚规划（摘要）

**F407 引脚分配要点**：
- CAN1 走 PB8/PB9 remap（避开 USB 和 FSMC）
- 外置 W25Q64 走 SPI3（PC10/11/12，复用 SD 卡脚）
- AT24C02 走 I2C3（PA8/PC9）
- 板载 W25Q16 与 LCD 触摸共享 SPI1（PB3/4/5），不同 CS
- LCD 用 FSMC 完整 16 位 8080 接口
- 解决了 8 处潜在引脚冲突

**F103C8T6 引脚分配要点**：
- 编码器用 TIM2 编码器模式（PA0/PA1）
- PWM 用 TIM1_CH1（PA8）
- 方向/使能：PA4 (DIR) + PB0 (EN)
- CAN1 默认脚 PA11/12 无冲突
- USART1 调试 PA9/PA10
- LED PC13 + PB1
- 启动按键 PB2（BOOT1 启动后作普通 GPIO）

### 4.5 电源树关键节点

| 节点 | 电压 | 电流 | 来源 |
|------|------|------|------|
| DC 适配器 | 12V | 5A | 外部电源 |
| BTS7960 VM | 12V | 2-5A | 适配器直通 |
| BTS7960 VCC | 5V | <100mA | DCDC |
| DCDC 5V | 5V | <1A | LM2596/MP1584 |
| 板上 3.3V | 3.3V | <500mA | AMS1117 |
| LCD 背光 | 3.3V/5V | <200mA | 视屏而定 |

**抗干扰要点**：
- 电机入口 1000μF 大电容 + 0.1μF 瓷片
- 编码器信号双绞线 + 100Ω 串联 + 0.1μF 滤波
- DCDC 用非隔离 + π 型滤波（性价比）
- CANH/CANL 走双绞线，两端 120Ω 终端

## 七、关键决策记录

| 决策点 | 结论 | 原因 |
|--------|------|------|
| 主控架构 | F407 + F103 双 MCU | 用户已有资源；贴近真实 ECU |
| 通信 | CAN 500kbps | v1.0 共识，贴近车载 |
| 屏幕 | 3.5寸 8080 ILI9488 | F407 FSMC 性能匹配；触摸可后加 |
| 电机 | JGA25-370 有刷直流 | 2208 套件复杂度高，第一版避免 |
| 驱动 | BTS7960 | 比 TB6612 抗造 |
| OTA 方式 | YMODEM + W25Q64 | 不依赖网络，最稳 |
| RTOS | F407 用 FreeRTOS | 多任务调度需要 |
| CAN1 脚 | PB8/PB9 remap | 板上 PA11/12 被 USB 占用 |
| 外置 W25Q64 脚 | SPI3 (PC10/11/12) | 借用 SD 卡脚，SD 卡不焊 |
| AT24C02 脚 | I2C3 (PA8/PC9) | 避开 CAN1 和 FSMC |
| 板载 W25Q16 用途 | 字库+图标 | 充分利用 2MB 板载 Flash |
| 外置 W25Q64 用途 | OTA 暂存 + 资源 | 8MB 充裕 |
| 电源方案 | 12V/5A + LM2596 + 板上 AMS1117 | 简单可靠 |
| 隔离策略 | 非隔离 DCDC + π 型滤波 | 性价比 |
